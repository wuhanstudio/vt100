## VT-100

RT-Thread 的终端 msh 一直是一个非常受大家欢迎的组件，但是终端只能打印基本的字符未免有些单调了，于是我封装了 ANSI Escape 标准，这样就可以在 msh 里实现更加丰富的图形界面了，例如动画和游戏开发。
